#### 成就
1. 业务型成就
    1. 业务目标（公司核心目标，目标转指标）eg: 点击率-加入滑动左右切tab提升3倍
    2. 技术方案（业务指标到技术指标，形成纸面方案，完成小规模试验）
    3. 实施方案 (确定实施目标、参与人，管理实施进度)
    4. 结果评估 (数据采集、数据报表，向上级汇报)
2. 工程型成就 eg: xss攻击防范
    1. 目标：质量、效率（重要） 
    2. 方案与实施 （规章制度、库、工具、系统）
    3. 结果  线上监控
3. 技术难题
    1. 目标 公认的技术难题
